# Cursor Claude 웹사이트

"왜 Cursor Claude를 써야 하는가?"를 설명하는 웹사이트 프로젝트입니다.

## 파일 구조

```
Cursor_Claude_웹사이트/
├── index.html          # 메인 웹사이트 파일
└── README.md          # 프로젝트 설명
```

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 웹 서버에 배포하여 사용

- Cursor Claude의 핵심 장점 설명

- 실제 사용 사례
